# Summary

The School Rankings aims to streamline the admission process of pre-primary, primary and secondary schools of India with the help of technology. It converges the best of today's technology and promises to bring upon a change through convergence into the Indian education system by leveraging the conventional process of offering admissions into schools.

## Path Configuration  

In case, the paths of above prerequisites softwares are not configured on System Properties>Environment Variables of your machine, kindly follow the path configurations of the same:

- [Node JS Path Configuration](#nodejs-path-config) Copy folder path of node js post-installation.  
For Windows:
            C:\Program Files\nodejs\;

- [Git Path Configuration](#git-bash-path-config)
For Windows:
C:\Program Files\Git\cmd;
C:\Program Files\Git\mingw64\bin;
C:\Program Files\Git\usr\bin
